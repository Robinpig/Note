## Introduction

[**OpenCode**](https://opencode.ai/) 是一个开源的 AI 编码代理。它提供终端界面、桌面应用和 IDE 扩展等多种使用方式





## Installation



Homebrew

```shell
brew install anomalyco/tap/opencode

opencode --version
```



Windows 通过 `choco` 安装

```shell
choco install opencode

opencode --version
```



启动

```shell
opencode
```





## Model

Opencode 与传统 IDE 最大的不同在于它引入了 **Plan (规划)** 与 **Build (构建)** 两种工作模式
