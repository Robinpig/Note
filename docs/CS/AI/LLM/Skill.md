## Introduction

Skill 是什么？

一个 Skill 通常包含：

```
skill-name/
├── SKILL.md        # 核心说明（必须）
├── scripts/        # 可执行脚本
├── references/     # 文档或知识
├── assets/         # 附加资源
```
