## Introduction

[Ehcache](https://www.ehcache.org/) 是一个开源的、基于标准构建的缓存，可以提升性能、减轻数据库负载并简化可扩展性。
它是最广泛使用的基于 Java 的缓存，因为其健壮、久经考验、功能完备，并且能够与其他流行的库和框架集成。
Ehcache 从进程内缓存扩展到混合进程内/进程外部署，支持 TB 级别的缓存。

多级缓存 结合MyBatis

可持久到磁盘

可以实现分布式缓存

允许注册事件侦听缓存

Terracotta Server 使用MQ做消息同步
