## 密码与权限



##### Reset Password

1. `shift` when starting to Advanced options for ubuntu
2. `e` at second recover mode 
3. Use `quiet splash rw init=/bin/bash` replace `ro recovery nomodeset *`
4. `passwd ["username"]` change password
