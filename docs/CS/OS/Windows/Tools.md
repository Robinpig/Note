## 编译器

[Mingw-w64](https://github.com/niXman/mingw-builds-binaries/releases) 是一组头文件、导入库、库和工具的集合，当与编译器工具链（如 GCC 或 LLVM）结合时，提供了构建原生 Windows 应用程序和库的完整开发环境。
Mingw-w64 是原始 mingw.org 项目的改进版，该项目旨在 Windows 系统上支持 GCC 编译器。它于 2007 年被分支以提供 64 位支持和更新的 API。此后获得了广泛使用和分发。

[Cygwin](https://cygwin.com) 是：
- 大量 GNU 和开源工具的集合，提供类似于 Windows 上的 Linux 发行版的功能。
- 一个 DLL（cygwin1.dll），提供大量的 POSIX API 功能。
