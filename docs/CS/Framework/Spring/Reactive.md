# Reactive

## Introduction

*构建为 Reactive Systems 的系统更加灵活、松耦合且[可扩展](https://www.reactivemanifesto.org/glossary#Scalability)。
这使得它们更易于开发且易于变更。
它们对[故障](https://www.reactivemanifesto.org/glossary#Failure)的容忍度更高，并且当[故障](https://www.reactivemanifesto.org/glossary#Failure)确实发生时，它们以优雅而非灾难的方式应对。
Reactive Systems 具有高响应性，为[用户](https://www.reactivemanifesto.org/glossary#User)提供有效的交互式反馈。*

[The Reactive Manifesto](https://www.reactivemanifesto.org/)

Reactive Systems 具备以下特性：

1. Responsive（响应性）
2. Resilient（韧性）
3. Elastic（弹性）
4. Message Driven（消息驱动）
