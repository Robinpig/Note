## Introduction

[Resilience4j](https://resilience4j.readme.io/) 是一个轻量级容错库，灵感来自 [Netflix Hystrix](/docs/CS/Framework/Spring_Cloud/Hystrix.md)，但专为函数式编程设计。

Resilience4j 提供高阶函数（装饰器）来增强任何函数式接口、lambda 表达式或方法引用，支持断路器、限流器、重试或舱壁功能。
