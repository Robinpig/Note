## Introduction

[Dragonfly](https://dragonflydb.io/) 是一个健壮、超快的内存数据存储，完全兼容 Redis™* 和 Memcached。
Dragonfly 可垂直扩展，在单个实例上处理数百万 QPS 和数百 GB 的内存。

多 CPU

具有相对精度的过期时间

原生 HTTP 控制台和 Prometheus 兼容指标

## Cache Management

Dragonfly 拥有一个统一的单一日适应缓存算法，该算法非常简单且内存高效。
